# SZR Trščar - Website

Professional website for SZR Trščar, a thatched roof specialist from Temerin, Serbia.

## About

SZR Trščar is a family-owned business specializing in traditional thatched roofs (trščani krovovi) since 2010. They cover all of Serbia and have an AA+ credit rating.

## Features

- Responsive design (mobile-first)
- Modern, clean aesthetic with natural color palette
- Smooth scroll navigation
- Testimonial slider
- Image lightbox gallery
- Viber/Phone floating buttons
- Contact form with Viber integration
- SEO optimized

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts (Poppins)
- Font Awesome icons
- Embedded styles and scripts (single file deployment)
